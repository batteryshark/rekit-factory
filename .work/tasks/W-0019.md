## Goal
Evolve Factory from a durable fan-out control plane into a bounded, evidence-driven research system that can pursue long investigations, test explicit hypotheses, prove consequential findings, improve from authorized calibration work, and involve the operator only at meaningful decision points.

## Requirements
- [ ] Durable project memory survives runs, processes, model context boundaries, and harness changes
- [ ] Hypotheses, attempts, contradictory evidence, validation work, and stop conditions are first-class state
- [ ] A finding cannot be promoted by model assertion alone; proof and independent reproduction are explicit gates
- [ ] Long-running campaigns have authorization, budget, novelty, retry, and termination ceilings rather than an unbounded loop
- [ ] Calibration produces comparable measurements and can propose, but never silently install, durable capabilities
- [ ] Operator notifications are exception-based, idempotent, redacted, and linked to canonical Mission Control state
- [ ] External ranges, schedulers, and workflow systems integrate through narrow contracts without becoming a second ledger or bypassing policy

## Acceptance Criteria
- [ ] An offline, owned fixture can run as a restart-safe multi-epoch campaign from initial hypotheses through independently reproduced proof
- [ ] The same run yields a content-addressed dossier, a deterministic Mission Control projection, and a bounded resume context
- [ ] A false-positive candidate and a non-generalizing technique are both rejected without operator cleanup or knowledge/tool mutation
- [ ] An out-of-scope action, exhausted budget, repeated no-progress epoch, and unavailable handoff destination all fail closed without losing state
- [ ] Every child task has focused deterministic tests and identifies any separately authorized live proof
