## Goal
Persist the bounded canonical campaign health facts Mission Control needs without deriving policy state in the API or browser.

## Requirements
- [x] Store content-bound coverage/outcome totals, novel-progress counts, retry/no-progress counters, phase, elapsed observation, and next-checkpoint expectation with each applied epoch recommendation
- [x] Rebuild and validate every health rollup from the campaign event history; expose only bounded problem counts/codes when replay is degraded
- [x] Preserve exact campaign/epoch/checkpoint/policy-input identity and reject cross-campaign or regressing facts
- [x] Bound retained history and public projection size while keeping current and comparison values

## Acceptance Criteria
- [x] Restart/crash fixtures reproduce the same rollup exactly once without rerunning work
- [x] Coverage, novelty, retry/no-progress, elapsed, and next-checkpoint values cannot be forged by API/browser callers
- [x] Large histories, corrupt replay, concurrent campaigns, and missing observations fail closed with bounded diagnostics

## Progress Log
- 2026-07-13T22:15:00.000Z — Split from W-0055 after implementation proved W-0054 durably retained recommendation identity and usage but not the policy-input health totals/counters needed for an honest operator projection.
- 2026-07-13T23:38:00.000Z — Added content-bound health events and a 32-observation projection, exact staged policy/recommendation recovery, state-aware operator-stop reconciliation, deterministic replay validation, bounded public diagnostics, migration-safe checkpoint sequencing, and explicit 256-fact policy-input limits. Focused campaign suite passed 81 tests.

## Completion Summary
Campaign health is now canonical campaign history rather than browser-derived state. Each
observation binds the exact campaign, epoch, checkpoint, policy input, recommendation, totals,
counters, phase, elapsed usage, and optional checkpoint expectation. Restart and operator-stop
races converge without rerunning work or losing a health observation after an effect begins;
the public projection remains bounded and excludes private policy envelopes.
