## Goal
Select or implement the subsystem that produces source-bound campaign risk measurements without deriving risk from prose or notification policy.

## Requirements
- [x] Define the authoritative risk producer and its bounded 0..100 semantics
- [x] Bind every measurement to an exact checkpoint source revision and digest
- [x] Call `CampaignController.record_risk_measurement` only after the producing fact is durably committed
- [x] Preserve source continuity, replay safety, restart recovery, and terminal-state rejection
- [x] Keep estimation evidence private while exposing only the canonical score/source identity

## Acceptance Criteria
- [x] Real campaign progress produces measurements without manual controller calls
- [x] Replayed or regressed source facts cannot change the measurement stream
- [x] Crossing the campaign contract threshold reaches the existing notification pipeline exactly once
- [x] No model prose, heuristic fallback, or missing fact can fabricate a score
