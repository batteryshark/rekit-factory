- [ ] Render only W-0054 canonical projections for epoch, elapsed/progress, novelty, coverage, budgets, retry/no-progress counters, scope, checkpoint, and terminal outcome
- [ ] Add pause/stop/resume and exact-content approval flows without letting the browser mutate canonical state directly

- [ ] Controls survive reconnect and repeated clicks without duplicate transitions
