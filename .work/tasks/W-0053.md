## Goal
Turn canonical evidence, hypothesis, coverage, finding, capability-gap, retry, and resource changes into a deterministic bounded continue/backoff/reprioritize/ask/stop policy.

## Requirements
- [x] Count progress only from new material evidence, hypothesis resolution, coverage movement, reproduced findings, or changed capability gaps—not model self-reports or repeated prose
- [x] Track cumulative and per-epoch ceilings exactly and reserve/refund spend without double counting retries or interrupted work
- [x] Detect repeated equivalent attempts, no novelty, validation churn, dependency deadlock, flapping environments, and notification churn
- [x] Produce reason-coded continue, reprioritize, backoff, suspend, ask-operator, exhausted, policy-stop, and success recommendations
- [x] Require exact durable approval before scope or hard cumulative ceiling increases
- [x] Remain pure under a fake clock with no model calls, hidden heuristics, randomness, or wall-clock dependence

## Acceptance Criteria
- [x] A no-novelty fixture asks or stops at the configured threshold without exhausting unrelated budget
- [x] A productive fixture advances recon→hypothesis→validation based only on canonical progress facts
- [x] Equivalent reordered inputs yield the same recommendation and explanation
- [x] Overflow, negative deltas, stale checkpoints, and contradictory totals fail closed
