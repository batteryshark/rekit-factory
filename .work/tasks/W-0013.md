## Goal
Support inspectable global provider/model profiles with safe per-run overrides and Mission Control selection.

## Requirements
- [ ] Secrets remain environment or keychain references and never enter run storage
- [ ] OpenAI-compatible endpoints such as MiniMax and LM Studio are first-class
- [ ] Profiles declare structured-output mode and concurrency/retry limits

## Acceptance Criteria
- [ ] The effective profile is visible before launch and persisted without credentials
- [ ] MiniMax M3 and a local compatible endpoint can each complete the smoke investigation

## Plan
Extend the profile schema with explicit structured-output behavior, concurrency, and retry limits; surface the effective values in Mission Control and persisted run metadata; complete the local OpenAI-compatible endpoint smoke.

## Notes
The baseline MINIMAX profile is implemented and live-tested; prompted JSON is required for reliable MiniMax M3 structured output.
