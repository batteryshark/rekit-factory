## Goal

A persistent, harness-neutral **project ledger** (file protocol) under `REKIT_HOME/projects/<id>/`:
every artifact with hash, kind, provenance, derivations, pending work, and findings.

## Acceptance Criteria

- [ ] Close and reopen a project — ledger state reloads losslessly.
- [ ] Run a second goal over the same target — zero re-derivation (content-addressed cache reuse).
- [ ] Every orchestration step appends a typed event; replaying the log reconstructs the ledger.
- [ ] Project lifecycle: open / list / resume; a project is keyed by target and revisitable.
