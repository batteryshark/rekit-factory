**Target repo:** rekit

## Context

- Epic **E6**. Plan: `parallax/docs/rekit-epic.md` §E6. Depends on E2 (T-049) and E5 (T-052).
- Workflows collapse into **goalpacks**: a prompt + requested capabilities + their **own** report
  shape. They vendor parallax's taxonomy but share no report library.

## Goal

Define the goalpack shape; reframe mcd / understand / agent-risk as goalpacks; kill the shared
`report_model` / `normalize_report`; prove a non-assessment (act-shaped) goal.

## Acceptance Criteria

- [ ] A goalpack declares intent + its own report renderer and runs on rekit.
- [ ] `report_model` / `normalize_report` are **deleted**; the ledger holds only the generic
      findings/artifact substrate.
- [ ] mcd / understand / agent-risk run as goalpacks (assessment-shaped).
- [ ] An **investigation** goal produces a narrative; an **action** goal (e.g. a binary patch)
      produces a verified derived artifact — proving goals aren't only reports.

## Notes

- The mcd analysis engine is parallax (E5); only its report/sections/adjudication is the goalpack.
